# Refresh Story Checklist

## Pre-Refresh
- [ ] Story file exists in implementation-artifacts
- [ ] Active sprint identified (or story specified)
- [ ] Previous stories in epic identified

## Analysis Complete
- [ ] Git history analyzed for previous stories
- [ ] Previous story files reviewed for patterns
- [ ] Current codebase state verified
- [ ] Architecture/PRD context loaded

## Refresh Executed
- [ ] Dev Notes updated with implementation context
- [ ] Dev Notes updated with codebase state
- [ ] Architecture alignment documented
- [ ] Testing requirements included
- [ ] User edits preserved (AC, Tasks)
- [ ] Development Record sections preserved

## Output
- [ ] Story file updated successfully
- [ ] Last Refreshed date added
- [ ] Summary provided to user
